# maven-search-cli

Maven search command line tool.

### How to Build

`go build`

### Installation

`go get -u github.com/rsrini7/maven-search-cli`

### Usage

```
Options:
  --keyword <string>  Search keyword
  --type              Build type: gradle, maven, or sbt (default "gradle")
  --rows              Number of rows (default "20")
```
